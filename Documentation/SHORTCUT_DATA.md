# Shortcut catalogue

`CorelCompanion/Resources/shortcuts.json` intentionally contains only demonstration records from the product brief. Before release, replace or verify every record against CorelDRAW 2026 for Windows and macOS. Do not infer the macOS shortcut by mechanically replacing Ctrl with Command.

The schema stores modifiers and keys structurally. Display strings are generated by `ShortcutFormatter`, allowing hundreds of entries without UI changes.
